## Features

- JSON formatter & Validator
- JWT decoder
- Unix Time Converter
- SSL Information
- Base64 Decoder/Encoder
- URL Decoder
- JSON to YAML
- YAML to JSON
- Diff viewer
- Remove duplicate lines
- SpreadSheet Comparison
- AWS Profile Manager
- JSON to Excel/CSV
- Status Tracker
- Regex Tester
- Merge Excel files
- [Request a Feature](https://github.com/AndresMorelos/devapp/issues/new)
